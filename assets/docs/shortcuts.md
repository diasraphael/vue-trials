shortcuts

Mac / Windows in VS code

alt + shift + f : Format code
cmd + shift + f : To open file search
cmd + shift + p : To open commnd palette
cmd + , / ctrl + , : To open the settings panel
cmd + p : To open files
cmd + b : toggle between sidebar and code screen
alt + right or left arrow : move between one word left or right
cmd + up or down arrow : move to top or bottom of the page
alt + up or down arrow : move line to top of above line

to select the content from left or right
cmd + shift + left or right arrow

To move to end or start of the line
cmd + left or right arrow

search for files
cmd + p

To close a file
cmd + w

Toggle between files(for forward and reverse)
ctrl + tab & ctrl + shift + tab

To Full code mode
cmd + k and then release both and click z

To toggle or switch between multiple screens in vs code
cmd + 1 or cmd + 2

In Browser
cmd + p : print html page to pdf
